## 开发流程

使用 [pnpm](https://pnpm.io/zh/)包管理

```shell
# 安装依赖
  pnpm i
# 启动项目
  pnpm run dev
```

## 提交流程

每人创建一个分支 每开发完一个功能/模块 提交

开发完一个功能 提pr 找其他人review代码

提交规范 feat/fix: 功能描述
例： feat: 单据自训练 开始训练弹窗
